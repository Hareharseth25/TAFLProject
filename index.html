Reference

Formal definitions, algorithms, and input format guide.

Input Format

One production per line. Use → or -> as the arrow. Separate alternatives with |. Use ε or eps for the empty string.

S → A B | a
A → B B | ε
B → b

Uppercase = Non-terminals. Lowercase = Terminals.

⚠️ Start symbol must appear as LHS of at least one rule.

Simplification Steps

1
Remove Null Productions
Find nullable variables (A ⇒* ε). For each production containing a nullable variable, generate all "with/without" combinations. Remove all A→ε rules (keep S→ε if S is nullable).
↓
2
Remove Useless Symbols
A symbol is useful only if it is both generating (can derive terminals) and reachable (accessible from start). Remove non-generating first, then unreachable.
↓
3
Remove Unit Productions
Compute the unit closure: all (A,B) pairs where A⇒*B via unit steps. For each pair, copy non-unit rules of B into A. Delete all unit rules.
